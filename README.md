| Column Name | Description |
| ----------- | ----------- |
| Year | NHS financial year (e.g., 2023/24) |
| Quarter | Time period within the year (Annual, Q1, Q2, etc.) |
| Period of coverage | Specific date range covered |
| Breakdown | Region-level breakdown (currently all 'England') |
| Level / Level description | Administrative level (e.g., England, NHS Trusts) |
| Indicator value | Readmission rate per 100,000 people |
| Lower CI / Upper CI | Confidence intervals for readmission rate |
| Standardised ratio | Adjusted readmission rate |
| Observed | Actual number of readmissions |
| Population | Total population for the area |
| Expected | Expected readmissions based on historical data |
| Percent unclassified | Missing/unclassified data percentage (mostly NaN) |
